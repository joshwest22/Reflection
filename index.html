Hardest Thing I've Learned

The hardest thing that I have learned so far was when and how to use the .map() feature. I also struggled to grasp how it worked, but eventually after enough attempts and help from Dr. Bradshaw I was able to realize that .map should be used in any scenario in which you want to transform an array of one type of data (e.g. numbers, objects, strings, etc.) into an array of another type. Once you understand the context in which it is used, the syntax is no problem as it is: 

originalArray.map(newArrayTransformationFcn+5)
Where everything inside of the parantheses represents a function that transforms your original data into the desired data in the new array.
